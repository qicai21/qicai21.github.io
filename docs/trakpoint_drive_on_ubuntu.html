ubuntu: trackpoint setting

Date 五 07 四月 2017 Tags ubuntu / thinkpad_trackpoint

using xinput command

what's xinput?

xinput is a utility to list available input devices, query information about a device and change input device settings.

more info about xinput

Change configure of TrackPoint

Get Device ID of TrackPoint

$xinput list | grep TrackPoint
# get output as follows
TPPS/2 IBM TrackPoint    id=14    [slave  pointer  (2)]

Then get setting value by this ID：

$ xinput list-props 14

Device 'TPPS/2 IBM TrackPoint':
Device Enabled (137):   1
Coordinate Transformation Matrix (139): 1.000000, 0.000000, 0.000000, 0.000000, 1.000000, 0.000000, 0.000000, 0.000000, 1.000000
Device Accel Profile (263): 0
Device Accel Constant Deceleration (264):   1.000000
Device Accel Adaptive Deceleration (265):   1.000000
...
# the value in row 5 and 6, it output as：
Device Accel Constant Deceleration (264): 1.000000 
Device Accel Adaptive Deceleration (265): 1.000000

其中括号里的整数就是该项设置的ID，后面的小数是该项设置的参数值。 ID为264（我的机器）代表的是小红点的固定阻力值(Constant Deceleration)， 取值范围是 [0,1]；ID为265代表的是随力度减弱时阻力增加的速度， 即小红点停下来的速度，取值范围是 [1,?]。一般来说设置264就够了.

reset the value:

$xinput set-prop 14 264 0.40

把下边这条命令加入到 ~/.xprofile

#!/bin/bash
# obtain TrackPoint ID from xinput list
TP_ID=$(xinput list | grep TrackPoint | cut -f 2 | grep -Eo '[0-9]{1,}')

if [ -n "$TP_ID" ]; then
    # obtain properties from xinput list-props "$TP_ID"
    AS_ID=$(xinput list-props "$TP_ID" | grep 'Accel Constant Deceleration (' | cut -f 2 | grep -Eo '[0-9]{1,}')
    # set the speed you want
    $(xinput set-prop "$TP_ID" "$AS_ID" 0.50 )
fi
