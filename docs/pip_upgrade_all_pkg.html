一条命令实现pip更新全部包

Date Tue 29 May 2018 Tags ubuntu / python3

通过下边的命令可以更新出全部pip包,但因为有破坏不同包的依赖性问题,所以谨慎使用.

$ pip freeze --local | grep -v '^\-e' | cut -d = -f 1 \
| xargs pip install -U
